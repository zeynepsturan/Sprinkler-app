A specialized mobile application built to ensure the survival and health of newly planted trees by tracking hydration schedules and monitoring growth through an interactive map.

---

## ðŸš€ Overview
**Sprinkler** is a community-driven designed to solve the problem of forgotten saplings. It allows users to pinpoint trees on a map, keep a rigorous log of when they were last watered, and set reminders to ensure no tree is left to dry out.

> ðŸ”’ **Note:** The source code for this project is private. This repository serves as a functional showcase of the project's UI/UX and technical implementation.

---

## âœ¨ Key Features
* **Interactive Tree Mapping:** Pinpoint tree locations to ensure they are easily found and cared for in the field.
* **Community Feed & Announcements:** A dedicated space for community-wide announcements, news, and member posts to keep everyone engaged.
* **Tree-Specific Social Feeds:** Every tree has its own comment section, allowing users to share updates, photos, and status reports directly on that tree's profile.
* **Hydration Tracking:** A simple logging system to record watering history and prevent dehydration.
* **User-Friendly Interface:** A minimalist and modern mobile experience built with React Native.
